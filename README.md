It has a Custom Driver (not really optimized because stills uses the TI original MSP430 header, to keeps compatibility betweens multiple processors.

- [x] GPIO
- [ ] GPIO Interrupts
- [x] WTDG
- [x] TIMERS
- [ ] UART
- [ ] ADC
- [ ] LCD
